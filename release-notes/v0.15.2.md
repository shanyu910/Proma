# Proma v0.15.2 更新

## 新功能

- **查看 Codex OAuth 订阅额度** — 在 Agent 上下文用量和渠道卡片中展示 ChatGPT Codex 的滚动额度、剩余百分比与重置时间，支持 5 小时、每周及其他实际限额窗口。

## Bug 修复

- **修复 Codex OAuth 代理支持** — 授权码换取与刷新 Token 请求现在复用 Proma 的全局代理设置，网络受限环境中也能完成登录和续期。
- **修复飞书 Pi 流式消息重复/乱序** — 正确处理 Pi 的累计 partial 帧并按内容块计算增量，避免卡片中的文本、思考内容和工具状态重复或错位。
- **修复历史记录中的相对路径预览** — 为历史工具调用补充会话工作目录作为解析上下文，使 `.context/plan/*.md` 等相对路径能正确打开。
